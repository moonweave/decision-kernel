# Skill Catalog

## anneal

Purpose: choose between competing implementation approaches by drafting a cheap
fitness measure, scoring candidates, and stopping before expensive build work.

Maturity: highest of the three. It has rubrics, assets, evaluation fixtures, and
the clearest product shape.

Productization verdict: viable as the lead skill if packaged with examples and
before/after decision reports.

## compass

Purpose: audit whether a long agent session has drifted from its original
intent and whether the local codebase has accumulated rot.

Maturity: useful internally after the portability hardening. It still needs more
real-session examples before it is credible externally.

Productization verdict: useful as part of an agent-ops bundle, not as a
standalone paid product yet.

## decide

Purpose: decide or escalate technical choices using local project preflight and
current source-backed evidence.

Maturity: strong as a policy gate. Its value depends on disciplined source use
and the agent honoring the escalation boundary.

Productization verdict: best bundled as a decision governance skill rather than
sold alone.
