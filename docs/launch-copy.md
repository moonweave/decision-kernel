## Hacker News / Reddit

```text
Show HN: Decision Kernel - Claude/Codex skills for agent judgment calls

Coding agents are usually good at executing once a direction is chosen. The weak
point is often the judgment boundary: choosing the wrong UI direction, drifting
away from the original task during a long session, or making a technical choice
from stale assumptions.

Decision Kernel packages three local SKILL.md protocols:

- anneal: turn competing approaches into a cheap measurable comparison
- compass: audit session drift and repository rot before continuing
- decide: make evidence-backed technical decisions from local context plus current sources

It is local-first, works as Claude Code skills and Codex skills, and the installer
is dry-run by default.

https://github.com/moonweave/decision-kernel
```
